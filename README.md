 BMI Calculator Web Application
BMI Calculator คือเว็บแอปพลิเคชันสำหรับคำนวณดัชนีมวลกาย (Body Mass Index) ที่ออกแบบมาให้ใช้งานง่าย มีความสวยงามทันสมัย รองรับการแสดงผลบนมือถือ (Responsive) พร้อมฟีเจอร์วิเคราะห์สุขภาพเบื้องต้นและกราฟแสดงแนวโน้ม
ฟีเจอร์หลัก (Key Features)
การคำนวณที่แม่นยำ: คำนวณค่า BMI ทันทีเมื่อกดปุ่ม
รองรับ 2 หน่วยวัด: เลือกใช้งานได้ทั้งแบบ Metric (kg/cm) และ Imperial (lbs/inches)
 กราฟและประวัติ (History & Trends):
บันทึกประวัติการคำนวณล่าสุด
แสดงกราฟเส้น (Line Chart) เพื่อดูแนวโน้มการเปลี่ยนแปลงของ BMI (ใช้ Library Chart.js)
การแสดงผลแบบ Visual:
แถบสีแสดงระดับ BMI (Scale Indicator)
แบ่งเกณฑ์ชัดเจน (ผอม, ปกติ, ท้วม, อ้วน)
 คำแนะนำสุขภาพส่วนบุคคล (Personalized Tips):
แนะนำด้านโภชนาการตามผลลัพธ์ BMI
คำแนะนำการออกกำลังกาย
คำนวณปริมาณน้ำที่ควรดื่มต่อวันตามน้ำหนักตัว
Responsive Design: รองรับการใช้งานทั้งบนคอมพิวเตอร์ แท็บเล็ต และสมาร์ทโฟน
 เทคโนโลยีที่ใช้ (Tech Stack)
HTML5: โครงสร้างหน้าเว็บ
CSS3: ตกแต่งความสวยงาม (Gradients, Grid Layout, Flexbox, Animations)
JavaScript (Vanilla): ประมวลผลลอจิกการคำนวณและการจัดการ DOM
Chart.js (CDN): ไลบรารีสำหรับสร้างกราฟแสดงผลข้อมูล
วิธีการติดตั้งและใช้งาน (Installation & Usage)
เนื่องจากโปรเจกต์นี้เป็นไฟล์ HTML แบบ Standalone คุณไม่จำเป็นต้องติดตั้ง Server หรือโปรแกรมเพิ่มเติม
ดาวน์โหลดไฟล์: บันทึกโค้ดเป็นไฟล์ชื่อ index.html หรือ bmiweb.html
เปิดใช้งาน: ดับเบิลคลิกที่ไฟล์เพื่อเปิดผ่าน Web Browser (เช่น Chrome, Edge, Firefox, Safari) ได้ทันที
การเชื่อมต่ออินเทอร์เน็ต: จำเป็นต้องเชื่อมต่ออินเทอร์เน็ตเพื่อโหลด Library Chart.js จาก CDN (สำหรับการแสดงกราฟ)
วิธีใช้งาน (How to Use)
กรอก ชื่อ (Name), อายุ (Age) และเลือก เพศ (Gender)
เลือกหน่วยวัดที่ต้องการ (Metric หรือ Imperial)
กรอก น้ำหนัก และ ส่วนสูง
กดปุ่ม "Calculate BMI"
ระบบจะแสดงผลลัพธ์ BMI, ระดับเกณฑ์, คำแนะนำสุขภาพ และอัปเดตกราฟด้านล่างสุดของหน้าจอ
สูตรการคำนวณ (Calculation Logic)
ระบบใช้สูตรมาตรฐานสากลในการคำนวณ:
BMI=ส่วนสูง(m)2น้ำหนัก(kg)​
(กรณีเลือกหน่วย Imperial ระบบจะแปลงค่าเป็น Metric ก่อนคำนวณ)
หมายเหตุ (Note)
ข้อมูลประวัติ (History) จะถูกบันทึกไว้ในหน่วยความจำชั่วคราว (Variable) เท่านั้น หากทำการ Refresh หน้าจอ ข้อมูลประวัติเก่าจะหายไป (ไม่ได้เชื่อมต่อฐานข้อมูล)
